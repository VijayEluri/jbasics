jbasics by schloepke

Generel purpose classes and helpers often helpful for java 5 and up

jBasics is a collection of classes often required in Java. The main goal is to have solutions for common tasks.

Dependencies

No Dependencies other than Java. Some code requires optional dependencies like java persistence.
